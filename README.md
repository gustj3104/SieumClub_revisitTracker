# Revisit Tracker

Small React app for tracking repeat attendance across sessions.

## Stack

- React 18
- Vite 6
- React Router 6
- Vitest
- IndexedDB

## Scripts

- `npm run dev`: start the Vite dev server
- `npm test`: run tests once with Vitest
- `npm run build`: create a production build in `dist`
- `npm run preview`: preview the production build locally
- `npm run deploy`: publish `dist` with `gh-pages`

## Deployment Base Path

The app is built with the GitHub Pages base path `/SieumClub_revisitTracker/`.

If the repository name changes, update:

- `homepage` in `package.json`
- `base` in `vite.config.mjs`
